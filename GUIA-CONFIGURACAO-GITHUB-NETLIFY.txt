CONFIGURAÇÃO CORRETA NO NETLIFY

Este projeto foi ajustado para funcionar com as variáveis do Netlify.
Não é mais necessário mostrar token no painel.

CADASTRE NO NETLIFY:
GITHUB_TOKEN=SEU_TOKEN
GITHUB_REPO=SEU_REPOSITORIO
GITHUB_BRANCH=main
SESSION_SECRET=SUA_CHAVE_SECRETA

DEPOIS:
1. Faça novo deploy
2. Entre com admin
3. Crie o usuário
4. Clique em "Subir pro GitHub"
5. Atualize a página
6. Teste o login do novo usuário

Se o GitHub estiver configurado, criar/editar usuário já salva no GitHub
e o botão "Subir pro GitHub" confirma a sincronização dos arquivos.
